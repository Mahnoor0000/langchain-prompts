# LangChain Prompts 

This is a **self-learning practice** where I explore how prompts, chatbots, and LLM workflows work using Python.

The goal of this project is to understand how to:
- design prompts
- structure conversations
- build simple chatbot logic
- experiment with LLM APIs
- organize prompt templates

---

## What I Learned

- Basics of prompt engineering
- How message history affects chatbot responses
- How to structure prompts dynamically
- How to organize LLM-related code into modules
